The Antagonist | Strategic Audit Tool

The Antagonist adalah chatbot cerdas yang dirancang khusus untuk menjadi teman brainstorming yang agresif. Berbeda dengan AI pada umumnya yang cenderung setuju dengan pengguna, bot ini menggunakan logika tingkat tinggi untuk mencari celah dalam rencana bisnis, strategi, atau argumen Anda.

Value utama dari alat ini adalah membantu eksekutif dan pengambil keputusan menghindari confirmation bias dengan memaksa mereka berpikir lebih tajam dan mengantisipasi kegagalan sebelum terjadi.

👥 Target Pengguna

Alat ini dirancang khusus untuk profesional yang membutuhkan opini kedua yang jujur dan tajam:

CEO & Eksekutif: Untuk menguji ketahanan strategi korporat sebelum eksekusi.

Entrepreneur & Founder: Untuk menemukan celah dalam pitch deck atau model bisnis sebelum bertemu investor.

Product Manager: Untuk membedah rencana peluncuran fitur dan memprediksi potensi penolakan pasar.

Konsultan Strategi: Sebagai alat simulasi debat untuk memperkuat argumen klien.

Pengambil Keputusan: Siapa pun yang ingin keluar dari "ruang gema" (echo chamber) dan mendapatkan audit objektif.

🚀 Fitur Utama

Analisis Agresif & Skeptis: Menggunakan model Gemini AI yang dikonfigurasi untuk memberikan kritik tajam dan logis.

Dukungan Multimodal: Dapat menganalisis teks, gambar (flowchart/diagram), dokumen PDF/Docx, hingga rekaman audio pitch.

Slider Intensitas: Atur level kekritisan sesuai kebutuhan audit Anda.

Antarmuka Profesional: Desain dark-mode premium dengan nuansa cyber-audit yang fokus pada produktivitas.

🌡️ Level Skeptisisme

Anda dapat mengatur tingkat "kekejaman" audit melalui slider intensitas:

Level 1-3: Pragmatis

Fokus pada kendala logistik dan efisiensi. AI akan memberikan kritik membangun yang berorientasi pada hasil praktis.

Level 4-6: Skeptis

AI mulai meragukan asumsi dasar Anda. Cocok untuk menguji validasi pasar dan proposisi nilai.

Level 7-8: Sangat Kritis

Mode audit mendalam. AI akan mencari celah sekecil apa pun dan mempertanyakan setiap data yang Anda berikan.

Level 9-10: Brutal / Penghancur

Mode tanpa ampun. AI akan mencoba meruntuhkan seluruh fondasi ide Anda untuk melihat apakah ada bagian yang benar-benar kuat untuk bertahan.

🛠️ Teknologi yang Digunakan

Frontend: HTML5, CSS3 (Custom Dark Theme), JavaScript (Vanilla).

Backend: Node.js, Express.js.

AI Engine: Google Gemini API (gemini-2.5-flash).

Icons: Lucide Icons.

📥 Instalasi

Clone Repositori

git clone [https://github.com/yapensa/the-antagonist.git](https://github.com/yapensa/the-antagonist.git)
cd the-antagonist


Instal Dependensi

npm install


Konfigurasi Environment
Buat file .env di direktori utama:

GEMINI_API_KEY=your_api_key_here
PORT=3000


Jalankan Aplikasi

npm start


Dibuat dengan intensitas tinggi untuk para pengambil keputusan yang berani menghadapi kebenaran pahit.
